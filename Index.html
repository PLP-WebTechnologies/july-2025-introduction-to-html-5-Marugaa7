Semantic landmarks

We use elements like headers, navigation, main and footer to help users understand the layout of the page. These tags act as landmarks fo assistive technology.

Attributes

Attributes such as aria-label enhance navigation by providing more context to screen readers. This comes in handy with users with disabilities
